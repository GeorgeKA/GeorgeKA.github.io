Education & Skills

Bachelor of Engineering - Mechatronics Engineering

In September 2010 I began my undergrad in mechatronics engineering at McMaster University.
After 4 years and internship at Blackberry I successfully completed the program in April 2015.

What is mechatronics engineering? Simply put, it is the design of computer controlled electromechanical systems.

Technical Skills

Over my time in the industry I've managed to develop a breadth and depth of skills.
Here are a couple highlights.

Over the years I've developed a strong ability in object oriented programming. While working on defect analysis in complex code bases I've had to use that knowledge heavily while debugging. When basic log file analysis was insufficient, I've leaned on tools like gdb to step through core file stacks and piece together the origins of crashes. When convenient cores weren't available, I've gone deeper into assembly language to get a sense of the low level instructions and registers involved in particular stack frames. When commands have performed unexpectedly on the command line, I've used strace to investigate the multitude of OS calls involved. In summary, I've become a problem solver.

System administration, in particular memory management has been a recurring theme in my work. Though my work was on the development side, I've been involved in many customer situations where memory management was crucial, especially on systems that run particularly resource intensive applications that compete with each other. I have the know-how to investigate the breaking points of systems, then provide recommendations to get users back on their feet.

In my earlier years I developed features for Blackberry 10 OS, and robot control software using Player Stage.

To complement mechatronics engineering, I've got quite a bit of C experience from implementing various applications using microprocessors (ex. ARM), in conjunction with my object oriented work.
Some of those applications include: a temperature sensing fan, stepper motor control, CAN communication, and a seven segment display.

Most of my work has been on unix based operating systems, though I've worked on teams that supported Windows as well.

The website that you're currently perusing was developed by myself, hence the web development experience.
Also, much of the functionality of the infection tracking project was written with Javascript, and styled with CSS and HTML.

Photo editing and graphic design (by way of Adobe Photoshop) have been personal interests of mine for some time. I have a collection of projects available to view.

Relational database management systems are at the heart of major data storage systems.
Beyond working on Db2 for some time, I am certified in its use. IBM Certified Database Associate - DB2 10.1 Fundamentals

Version control is key to software development productivity. To this end, I've used GitHub with multiple teams, as well as Zenhub.
Additionally, this website is hosted with the help of GitHub.
